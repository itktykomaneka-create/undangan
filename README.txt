SISTEM UNDANGAN DIGITAL
=======================

File yang ada di ZIP ini:
- undangan.html   → Halaman undangan (untuk tamu)
- dashboard.html  → Dashboard pengelola (untuk kamu)
- apps-script.js  → Kode backend Google Apps Script
- panduan.html    → Panduan lengkap setup & penggunaan
- images/         → Folder untuk foto-foto
- music/          → Folder untuk musik latar (opsional)

LANGKAH CEPAT:
1. Buka panduan.html di browser untuk panduan lengkap
2. Setup Google Apps Script (ikuti Bagian 03 di panduan)
3. Edit undangan.html (cari komentar GANTI:)
4. Uji coba di localhost (ikuti Bagian 05 di panduan)
5. Deploy ke GitHub Pages (ikuti Bagian 07 di panduan)
